# todo-backend
API server for todo web application


## how to run it

### 1. install openjdk11
- install [openjdk11](https://jdk.java.net/java-se-ri/11)

### 2. run server

#### with intellij
- click `Run 'TodoApplication'` button

#### with command line
```bash
./gradlew bootRun
```

### swagger-ui
- local: http://localhost:8088/swagger-ui.html
